10流程图


回到顶部

流程图分为竖向和横向两大类，竖向包括自上而下和自下而上两种顺序，横向包括从右到左和从左到右两种顺序。
对应语法分别为：graph TB/graph BT/graph RL/graph LR。

  • TB - top bottom（自上而下）

  • BT - bottom top（自下而上）

  • RL - right left（从右到左）

  • LR - left right（从左到右）
